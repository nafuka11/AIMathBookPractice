## 必要物
- Poetry
- Python >= 3.8
- [MeCab](http://taku910.github.io/mecab/)

## 実行方法
必要なパッケージのインストール
```bash
poetry install
```

JupyterLabの起動
```bash
poetry run jupyter-lab
```
